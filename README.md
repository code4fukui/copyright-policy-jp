# copyright-policy-jp

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

This repository archives and tracks the development of open data standards and government data usage policies in Japan.

## Policy Documents

This repository contains the following key policy documents:

### Government Standard Terms of Use (政府標準利用規約)

*   [Version 2.0](政府標準利用規約(第2.0版).md)
    *   Compatible with Creative Commons Attribution 4.0 International (CC BY 4.0).
*   [Draft Version 3.0](政府標準利用規約(第3.0版)案.md)
    *   Compatible with CC BY 4.0, Open Data Commons Attribution License 1.0 (ODC-By 1.0), and Open Database License 1.0 (ODbL 1.0).

### Open Data Standard Terms of Use (オープンデータ標準利用規約)

*   [Version 1.0](OpenDataStandardTermsofUse/)
    *   Compatible with CC BY 4.0. Also permits use under ODC-By 1.0 or ODbL 1.0.

---

**Note:** Under these policies, simple artifacts such as numerical data, tables, and graphs are not subject to copyright and can be used freely.
