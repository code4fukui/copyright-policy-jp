# copyright-policy-jp

このリポジトリは、日本のオープンデータ標準および政府データ利用方針の策定過程をアーカイブし、追跡しています。

## ポリシー文書

このリポジトリには、以下の主要なポリシー文書が含まれています。

### Government Standard Terms of Use (政府標準利用規約)

*   [第2.0版](政府標準利用規約(第2.0版).md)
    *   Creative Commons Attribution 4.0 International (CC BY 4.0) と互換性があります。
*   [第3.0版案](政府標準利用規約(第3.0版)案.md)
    *   CC BY 4.0、Open Data Commons Attribution License 1.0 (ODC-By 1.0)、および Open Database License 1.0 (ODbL 1.0) と互換性があります。

### Open Data Standard Terms of Use (オープンデータ標準利用規約)

*   [第1.0版](OpenDataStandardTermsofUse/)
    *   CC BY 4.0 と互換性があります。また、ODC-By 1.0 または ODbL 1.0 での利用も許可されています。

---

**注記:** これらのポリシーの下では、数値データ、簡単な表・グラフ等は著作権の対象ではなく、自由に利用できます。
